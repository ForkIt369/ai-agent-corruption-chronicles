## How to Use These Icons

1. Create each SVG file in the `public/icons/` directory with the filename specified in the headers.
2. You can use a script to automate this process, or create each file manually.
3. Once created, the icons will be available for use in the application.

  },

    name: 'trust.svg',

  <path d="M9 12l2 2 4-4" />

  },
  {
